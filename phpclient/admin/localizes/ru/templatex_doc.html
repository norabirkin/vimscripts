* - Файл шаблона должен удовлетворять следующим условиям:
  • Формат файла - HTML-текст
  • Максимальный размер файла - 10Кб
  • HTML текст файла должен представлять собой фрагмент web-страницы, который можно вставить в ячейку таблицы
  • HTML-текст шаблона может содержать следующие ключевые слова:
    Основные ключевые слова
    {contract_no} № договора
    {linkdate} Дата запланированного подключения
    {fio} ФИО абонента/Название юр. лица
    {address} Адрес абонента в формате "город, улица, дом, корпус, квартира, примечание"
    {phone} Телефон абонента
    {comments} Примечание
    {login} Логин
    {password} Пароль
    Доп. ключевые слова для формирования адреса
    {country} Страна
    {city} Город
    {street} Улица
    {house} Дом
    {building} Корпус
    {apt} Квартира
    {address_comments} Дополнительно
