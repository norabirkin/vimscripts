* - Файл шаблона должен удовлетворять следующим условиям:
  • Формат файла - HTML
  • Размер файла не должен превышать 300 кб
  • HTML текст файла может содержать ключевые слова из следующего списка:
    Общие ключевые слова
    {company_agent_fio} ФИО представителя компании, оформляющего договор (например, Иванов И.И.)
    {company_agent_name} Полное имя представителя компании, оформляющего договор (например, Иванов Иван Иванович)
    {contract_no} № договора
    {contract_date} Дата заключения договора
    {agent_no} ID агента (не конвергентный)
    {userlogin} Логин пользователя
    {userpass} Пароль пользователя
    {username} ФИО пользователя
    {userdescription} Описание пользователя
    {login} Логин (не конвергентный)
    {password} Пароль (не конвергентный)
    {email} Адрес электронной почты
    Доп. ключевые слова для юр лиц
    {company_fullname} Имя компании с формой собственности
    {address_u} Юридический адрес
    {gen_dir_u} Генеральный директор
    {gl_buhg_u} Гл. бухгалтер
    {kont_person} Контактное лицо
    {bank_name} Название банка
    {inn} ИНН
    {kpp} КПП
    {settlement_account} Расчетный счет
    {correspondent_account} Корр. счет
    {bic} БИК
    {address} Физический адрес
    {tariff_name} Название тарифа (не конвергентный)
    {act_on_what} На основании чего действует
    Доп. ключевые слова для физ лиц
    {fio} ФИО абонента (например, Иванов И.И.)
    {account_name} Полное имя абонента (например, Иванов Иван Иванович)
    {passport_number} Серия и номер паспорта
    {passport_issue_data} Кем, когда выдан паспорт
    {phone} Телефон
    {address} Адрес проживания
    {address_real} Адрес прописки
    {tariff_name} Название тарифа (не конвергентный)
    {inn_fiz} ИНН
    Доп. ключевые слова для юридического адреса юр. лиц или адреса проживания физ. лиц
    {country_u} Страна
    {city_u} Город
    {street_u} Улица
    {house_u} Дом
    {building_u} Корпус
    {apt_u} Квартира
    {address_comments_u} Дополнительно
    {region_u} Регион
    {district_u} Район
    {settle_area_u} Населенный пункт
    Доп. ключевые слова для физического адреса юр. лиц или адреса прописки физ. лиц
    {country} Страна
    {city} Город
    {street} Улица
    {house} Дом
    {building} Корпус
    {apt} Квартира
    {address_comments} Дополнительно
    {region} Регион
    {district} Район
    {settle_area} Населенный пункт
    Доп. ключевые слова для Доп. соглашений
    {add_date} Дата заключения доп. соглашения (не реализовано)
    {tariff_change_date} Дата смены тарифного плана (не конвергентный)
    {new_tariff_name} Название нового тарифа (не конвергентный)
    {ip_transport} IP/Маска учетной записи пользователя (Если несколько, то перечисление будет через ',')
