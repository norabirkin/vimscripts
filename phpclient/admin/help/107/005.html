Отчет по начислениям за услуги интернет базового направления

Формируемый документ отображает информацию по пользователям с которых было произведено списание денежных средств по трафику не входящему в тарифицируемые зоны IP каталога, а также списания по АБ плате.
Пользователи, у которых не было трафика и списаний - игнорируются.
Данные агрегируются только по кабельным агентам исключая Radius.

Отчет, формируемый менеджером, не включает удаленных пользователей!
