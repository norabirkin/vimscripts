Отчет по списаниям средств

Данных отчет формируется на основании записей в таблице «расходов» АСР. Включает в себя информацию исключительно по абонентам у которых были списания по абонентской плате, либо расход трафика и списание по нему. Абоненты у которых не было ни расхода ни списаний в отчет не включаются. Колонки «Включено в аренду» и «Превышение» не суммируются, поскольку могут содержать разные величины измерения для одного пользователя по тарифам. Для учетных записей агента IVOX указывается количество раз оказаных разовых и периодических услуг.

Отчет, формируемый менеджером, не включает удаленных пользователей!
