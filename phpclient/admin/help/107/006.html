Отчет по начислениям за услуги интренет по категориям IP каталога

Формируемый документ отображет информацию по пользователям с которых было произведенно списание денежных средств только по трафику совпашему с назначеной зоной IP каталога.
Пользователи, у которых не было трафика и списаний - игнорируются.
Данные агрегируются только по кабельным агентам исключая Radius.

Отчет, формируемый менеджером, не включает удаленных пользователей!
